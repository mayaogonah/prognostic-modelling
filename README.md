# Violent victimisation after adolescent traumatic brain injury: development and validation of a clinical prediction model

R/Quarto repository for the manuscript 'Violent victimisation after adolescent traumatic brain injury: development and validation of a clinical prediction model' which is currently in press at _Injury Prevention_.

The reposity contains the code to develop and internally validate a survival prognostic model to predict the risk of violent victimisation following adolescent traumatic brain injury. [The HTML file](https://mayaogonah.github.io/prognostic-modelling/) rendered from Quarto, outlines the key stages of analysis. 

This study makes use of anonymised data held in the Secure Anonymised Information Linkage (SAIL) Databank. We would like to acknowledge all the data providers who make anonymised data available for research. SAIL Databank prohibits individual-level data from being publicly available. Researchers who are interested in replicating this study can apply to work with the data through [SAIL Databank](https://saildatabank.com/data/apply-to-work-with-the-data/). The study protocol was pre-registered on [SAIL Databank’s Data Use Register](https://saildatabank.com/data/data-use-register/) (project code 1478).
