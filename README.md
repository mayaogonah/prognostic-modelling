**Overview**

This repository includes a HTML file (https://mayaogonah.github.io/prognostic-modelling/) created with Quarto, which details the code used to develop and internally validate a survival prognostic model to predict the risk of violence-related adverse outcomes following adolescent TBI. Please note that this is an **ongoing research project**. The findings presented have not yet undergone peer review or been published in a scientific journal.
